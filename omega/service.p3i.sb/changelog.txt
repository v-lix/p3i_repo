v0.1.3
- Fix: use "VideoPlayer.AudioCodec" infolabel instead of the bogus
  "Player.AudioCodec". The old label name didn't resolve, so Kodi
  echoed the literal string back, which never matched TrueHD/EAC3 and
  prevented the curated JSON-list path from engaging. The marker-file
  path was unaffected (it skips the codec gate).

v0.1.2
- Force-engage when a file named "SB" or "SB.txt" is present in the
  same folder as the playing file. Works in addition to the curated
  IMDb/TMDB list, so users with missing or unscraped metadata can
  opt-in per-folder without waiting for a list update. The marker
  bypasses the codec gate -- user said engage, we engage.
- New service-addon icon.

v0.1.1
- Add "show notification when LAV mode is switched" setting (default on)
  so users can see at a glance that the helper engaged on an SB title.

v0.1.0
- Initial release.
- Auto-toggles coreelec.amlogic.dolbyvision.audio.seamlessbranch on AV-start
  for movies in the bundled curated list whose audio is TrueHD or EAC3.
- IMDb and TMDB ID matching against resources/seamless_branching.json.
- Optional user additions via <profile>/seamless_branching_user.json.
- Defers to PlexMod4Kodi (script.plexmod) when it is the active session.
